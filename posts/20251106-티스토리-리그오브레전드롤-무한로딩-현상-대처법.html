리그오브레전드(롤) 무한로딩 현상 대처법
티스토리🗓 2025-11-06

최근 리그오브레전드(League of Legends) 실행 시
게임 시작 단계에서 무한로딩이 발생하는 현상이 다수 보고되고 있습니다.
해당 문제는 TouchEn 보안모듈과의 충돌로 확인되었습니다.

1. 문제 현상

  • 게임 클라이언트 실행 후 챔피언 선택까지는 정상 진행
  • 게임 시작 직후 화면이 멈추거나 로딩에서 진행되지 않음
  • 작업관리자(Task Manager)에서 CPU 사용률이 90~100%로 상승
  • “LeagueClientUxRender.exe”, “RiotClientServices.exe” 등의 프로세스가 비정상적으로 자원 점유

⚙️ 2. 주요 원인

원인은 TouchEn 계열 보안프로그램의 입력 훅(Hook) 충돌로 분석됩니다.

프로그램명주요 기능문제 원인
TouchEnKey.exe 키보드 보안 게임 클라이언트의 입력 감지와 중복 훅 발생
TouchEn nxClient.exe 통합보안 관리 프로세스 감시 기능이 클라이언트와 충돌
TouchEn mVaccine 보안 점검 백그라운드 검사 중 CPU 자원 과다 사용

TouchEn은 금융 및 공공기관 사이트 접속 시 자동으로 설치되는 보안모듈이며,
게임, 브라우저, 일부 개발 프로그램과의 충돌 사례가 빈번합니다.

3. 해결 방법

✅ 1단계: 보안모듈 삭제

  1. 제어판 → 프로그램 추가/제거 (앱 및 기능)
  2. 아래 프로그램 검색 후 제거
    • TouchEnKey, TouchEn nxClient, TouchEn mVaccine
  3. 삭제 완료 후 롤(League of Legends) 재실행

※ 별도의 재부팅은 필요하지 않음
삭제 즉시 정상 동작이 확인됩니다.

✅ 2단계: 시작프로그램 비활성화(선택)

  1. Ctrl + Shift + Esc → 작업관리자 실행
  2. 시작프로그램 탭에서 TouchEn, AnySign, AhnLab Safe Transaction 등 자동실행 항목 비활성화

⚠️ 4. 주의 사항

  • TouchEn은 금융·공공 사이트 접속 시 자동 재설치될 수 있습니다.
  • 향후 동일 증상 발생 시 동일한 절차로 삭제 필요
  • 삭제 시 은행 또는 정부24, 홈택스 등 일부 사이트에서 입력창 오류가 발생할 수 있으므로
    필요 시 재설치 후 사용 완료 즉시 제거 권장
