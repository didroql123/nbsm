엑셀 파워쿼리로 코인 시세 가져오기
티스토리🗓 2025-11-24

회사에서도 눈치 안 보고 엑셀 켜놓고 코인 시세를 확인할 수 있다면?
오늘은 엑셀 파워쿼리(Power Query) 로 코인 시세를 실시간으로 불러와
“일하는 척하면서 몰래 코인 보는 법”을 소개합니다

엑셀 파워쿼리 활용 요약

  • API 연결: 업비트 / 빗썸 / 바이낸스 / 환율 API 불러오기
  • 자동 계산: 실시간 시세 + 김프(%) 계산
  • 시각화: 조건부 서식으로 최고가·최저가 강조
  • 자동 새로고침: 1분마다 시세 갱신
  • 응용 가능: ERP / 재고 / API 데이터 자동화까지 확장 가능

 

 

실전 가이드 (고급편집기 코드 포함)

엑셀에서 파워쿼리 고급 편집기를 활용해
한 번에 10개 코인 데이터를 불러오는 코드까지 모두 정리했습니다.
