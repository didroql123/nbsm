게임 엔진의 종류
티스토리🗓 2025-02-17

게임 개발에 사용되는 대표적인 엔진들을 종류별로 정리해보았습니다.

1. 멀티플랫폼 지원 엔진

  • Unreal Engine: AAA급 게임 개발에 적합한 강력한 엔진.
  • Unity: 2D 및 3D 게임 개발이 가능한 인기 엔진.
  • Godot: 오픈 소스로 가벼운 2D 게임 개발에 최적화된 엔진.

2. 2D 게임 특화 엔진

  • GameMaker Studio: 간단한 2D 게임 제작에 최적화.
  • RPG Maker: RPG 장르에 특화된 엔진.
  • Construct: 코드 없이 2D 게임을 제작할 수 있는 비주얼 스크립팅 엔진.

3. 특정 장르에 최적화된 엔진

  • CryEngine: 리얼리틱한 그래픽을 필요로 하는 FPS 및 오픈월드 게임에 강점.
  • Ren'Py: 비주얼 노벨 및 인터랙티브 스토리텔링 게임 개발용 엔진.
  • Adventure Game Studio: 포인트 앤 클릭 어드벤처 게임 개발에 특화.

4. 웹 및 모바일 중심 엔진

  • Cocos2d-x: 모바일 게임 개발에 강한 오픈소스 엔진.
  • Phaser: HTML5 기반의 웹 게임 제작에 최적화.
  • GDevelop: 코드 없이 웹 및 모바일 게임 제작 가능.

5. 특수 목적 게임 엔진

  • Amazon Lumberyard: AWS와 연동이 강한 엔진.
  • Panda3D: Python 기반으로 개발된 오픈소스 엔진.
  • Stride (Xenko): C#과 .NET 환경에서 사용 가능한 게임 엔진.

각 엔진은 목적과 사용 용도에 따라 선택해야 합니다.
