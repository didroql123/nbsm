회사에서 가장 많이 터지는 IT 문제 TOP5
티스토리🗓 2026-03-27

회사에서 컴퓨터 좀 한다고 하면
어느 순간부터 이런 말 듣게 됩니다.

“이거 왜 안돼요?”
“컴퓨터 이상한데요?”

전산팀 입장에서는
이미 하루에 몇 번씩 겪는 익숙한 상황이죠.

그래서 오늘은
회사에서 실제로 가장 많이 터지는 IT 문제 TOP5를 정리해봤습니다.

1. 프린터 출력 안됨

가장 많이 듣는 말입니다.

“인쇄 눌렀는데 안 나와요”

대부분 원인은

  • 프린터 오프라인
  • 드라이버 문제
  • 스풀러 오류

입니다.

해결 방법

 

 

2. 크롬 보안 때문에 프로그램 설치 안됨

요즘 특히 많이 터지는 문제입니다.

“은행 프로그램 설치가 안돼요”

크롬 보안 정책 때문에

  • Veraport 설치 차단
  • 외부 앱 실행 차단

이 자주 발생합니다.

해결 방법 정리

 

️ 3. 원격 데스크톱 연결 오류 (CredSSP)

재택근무 / 외부 접속할 때 많이 발생합니다.

“원격 접속이 안돼요”

대부분 원인은

  • 보안 정책 업데이트
  • 인증 문제

입니다.

해결 방법

 

4. 모니터 화면 안나옴 (No Signal)

“컴퓨터 켰는데 화면이 안 나와요”

이건 거의 80% 확률로

  • 케이블 문제
  • 그래픽 출력 포트 문제
  • 전원 문제

입니다.

해결 방법

 

5. 컴퓨터 갑자기 느려짐

“오늘따라 컴퓨터가 너무 느려요”

원인은 대부분

  • 보안 프로그램 충돌
  • 백그라운드 프로세스
  • 업데이트 꼬임

입니다.
