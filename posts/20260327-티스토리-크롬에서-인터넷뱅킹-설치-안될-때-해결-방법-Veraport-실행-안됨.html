크롬에서 인터넷뱅킹 설치 안될 때 해결 방법 (Veraport 실행 안됨)
티스토리🗓 2026-03-27

서론

인터넷뱅킹을 이용하려고 할 때
보안 프로그램이 설치되지 않아 막히는 경우가 있습니다.

특히 최근에는 IBK 기업은행과 같은 금융 사이트에서
통합 설치 프로그램(Veraport)이 정상적으로 실행되지 않는 사례가 자주 발생하고 있습니다.

다운로드는 되지만 실행이 되지 않거나,
설치가 진행되지 않고 계속 “미설치” 상태로 남아 있는 경우라면
단순 오류가 아니라 브라우저 설정 문제일 가능성이 높습니다.

이런 증상이 나타난다면

다음과 같은 경우라면 이 글의 해결 방법이 해당됩니다.

  • 설치 파일을 눌러도 아무 반응이 없음
  • 프로그램이 실행되지 않음
  • 인터넷뱅킹에서 계속 “설치 필요” 상태
  • 다운로드는 되는데 실행이 안됨

대부분 동일한 원인입니다.

⚠️ 원인

최근 크롬 보안 정책이 강화되면서
웹사이트에서 PC 프로그램을 실행하려고 할 경우 기본적으로 차단됩니다.

특히 금융권에서 사용하는

  • Veraport (통합 설치 프로그램)
  • 인증서 관련 모듈
  • 키보드 보안 프로그램

이런 구성 요소들이 영향을 받습니다.

즉, 파일 문제가 아니라
브라우저에서 실행 자체를 막고 있는 상황입니다.

해결 방법

1. 주소창 왼쪽 아이콘 클릭

크롬 주소창 왼쪽의 아이콘을 클릭합니다.

2. “기기에 있는 앱” 허용 설정

설정 항목 중에서

기기에 있는 앱 → 허용으로 변경

이 옵션이 꺼져 있으면
보안 프로그램이 실행되지 않습니다.

3. 새로고침 후 다시 실행

설정 변경 후 페이지를 새로고침(F5)하고
다시 설치를 진행하면 정상적으로 실행됩니다.

참고 (실제 사례)

네이버 블로그에 실제 해결 사례 정리해둔 글

 

✅ 정리

인터넷뱅킹 설치가 안 되는 경우

  • 프로그램 문제 ❌
  • PC 문제 ❌
  • 크롬 설정 문제 ✅

“기기에 있는 앱 허용” 이 한 가지로 해결되는 경우가 대부분입니다.

앞으로 금융 사이트 이용 시 비슷한 문제가 계속 발생할 수 있으니
한 번 알아두면 바로 해결할 수 있는 유용한 방법입니다.
