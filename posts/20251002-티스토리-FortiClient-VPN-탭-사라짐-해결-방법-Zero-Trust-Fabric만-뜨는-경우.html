FortiClient VPN 탭 사라짐 해결 방법 (Zero Trust Fabric만 뜨는 경우)
티스토리🗓 2025-10-02

원격 접속(SSL-VPN)을 위해 FortiClient를 사용하다 보면,
어느 날 갑자기 VPN 탭이 사라지고 Zero Trust Fabric 화면만 뜨는 경우가 있습니다.
기존에는 잘 쓰던 환경인데 갑자기 VPN 메뉴가 보이지 않아 당황스러우셨을 거예요.

증상

  • FortiClient 실행 시 VPN 탭이 보이지 않고
    Zero Trust Telemetry / Zero Trust Fabric 등록 화면만 출력됨
  • 기존에 정상적으로 접속하던 SSL-VPN 연결 불가
  • 레지스트리 확인 시 vpn_tab_hidden 값이 1로 설정되어 있음

️ 원인

 

 

 

FortiClient는 설치 모드와 정책에 따라 VPN 탭을 숨길 수 있습니다.

  • 자동 업데이트나 정책 적용 과정에서 VPN 메뉴가 비활성화된 것
  • 레지스트리 상에서 vpn_tab_hidden = 1 로 변경되어 VPN 메뉴가 숨겨진 상태

✅ 해결 방법

1. 관리자 권한으로 레지스트리 수정

  1. regedit → 관리자 권한으로 실행
  2. 경로 이동:
  3.  
    HKEY_LOCAL_MACHINE\SOFTWARE\Fortinet\FortiClient\FA_ESNAC
  4. 오른쪽 값 중에서
    • vpn_tab_hidden → 0
    • ztna_tab_hidden → 0
      으로 수정
  5. FortiClient 종료 후 재실행 → VPN 탭 복구 확인

2. 명령어로 강제 수정 (관리자 CMD/PowerShell)

레지스트리 권한 문제로 직접 수정이 안 될 경우,
관리자 권한 CMD에서 아래 명령 실행:

 
reg add "HKLM\SOFTWARE\Fortinet\FortiClient\FA_ESNAC" /v vpn_tab_hidden /t REG_DWORD /d 0 /f
reg add "HKLM\SOFTWARE\Fortinet\FortiClient\FA_ESNAC" /v ztna_tab_hidden /t REG_DWORD /d 0 /f

3. FortiClient 재설치 (대안)

  • Fortinet 공식 사이트에서 FortiClient VPN 전용 버전 다운로드 후 재설치
  • VPN 전용 버전은 Telemetry/EMS 메뉴가 없고 VPN 메뉴만 표시됨

정리

  • 갑자기 VPN 탭이 사라진 이유는 FortiClient 설정(레지스트리)에서 VPN 메뉴가 숨김 처리 되었기 때문
  • vpn_tab_hidden 값을 0으로 변경하면 VPN 메뉴가 다시 표시됨
  • 재설치 대신 레지스트리 수정으로 간단히 해결 가능
