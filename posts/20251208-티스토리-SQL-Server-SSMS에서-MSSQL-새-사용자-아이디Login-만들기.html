SQL Server SSMS에서 MSSQL 새 사용자 아이디(Login) 만들기
티스토리🗓 2025-12-08

SSMS MSSQL 환경에서 새로운 사용자 아이디(Login)를 만드는 작업은 데이터베이스 보안과 관리의 핵심 요소입니다.

기업이든 개인 프로젝트든 사용자 권한을 올바르게 설정하지 않으면, 데이터 누락이나 권한 오남용 같은 문제로 이어질 수 있습니다.

​

예를 들어 개발자에게는 특정 DB 조회 권한만 필요한 반면, 운영 담당자는 전체 권한이 필요할 수도 있습니다.

이러한 상황에서 SSMS를 활용하면 간단하고 명확한 절차로 로그인을 생성할 수 있습니다.

​

이 글에서는

SSMS MSSQL 환경에서 새 로그인 생성 → 인증 방식 선택 → 데이터베이스 매핑 → 역할 멤버 자격 지정

까지 전체 과정을 쉽게 따라 할 수 있게 정리했습니다.
