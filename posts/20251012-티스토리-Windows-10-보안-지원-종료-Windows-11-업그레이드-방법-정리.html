Windows 10 보안 지원 종료, Windows 11 업그레이드 방법 정리
티스토리🗓 2025-10-12

2025년 10월 14일부로 마이크로소프트(Microsoft)의 Windows 10 보안 업데이트 지원이 공식 종료됩니다.
즉, 그 이후부터는 더 이상 보안 패치가 제공되지 않기 때문에 보안 위협에 그대로 노출될 수 있습니다.

그래서 이번 글에서는 Windows 11로 업그레이드하는 방법과 준비해야 할 사항을 간단하게 정리해 보겠습니다.

 업그레이드 전 준비사항

  • 데이터 백업 : 중요한 업무 문서나 개인 파일은 NAS, 외장하드, 클라우드 등에 꼭 백업하세요.
  • 전원 연결 : 설치 도중 전원이 꺼지면 오류가 발생할 수 있으므로 노트북은 반드시 충전기를 연결한 상태에서 진행해야 합니다.
  • 시간 확보 : 설치가 끝날 때까지 PC 사용이 불가하므로, 퇴근 이후 진행을 추천합니다.

 Windows 11 업그레이드 방법

  1. 작업표시줄 → 시작 → 설정 → 업데이트 및 보안 → Windows 업데이트
  2. “Windows 11 업그레이드 준비됨” 메시지가 표시되면 [다운로드 및 설치] 클릭
  3. 설치가 완료되면 자동으로 재부팅되며, Windows 11로 전환됩니다.

설치 불가 PC

일부 구형 PC는 Windows 11 권장 사양을 충족하지 못해 설치가 불가할 수 있습니다.
이 경우 업데이트 화면에 설치 불가 메시지가 나타납니다.

오류 발생 시 대처법

  • 설치 도중 에러 메시지가 뜨거나 중단될 경우,
    → 에러 코드 화면을 캡처해 두면 문제 해결에 큰 도움이 됩니다.

정리

  • Windows 10은 2025년 10월 14일 이후 보안 지원이 종료됩니다.
  • 보안과 안정성을 위해 Windows 11 업그레이드는 필수적입니다.
  • 백업 → 업데이트 확인 → 설치 및 재부팅 순서로 간단하게 진행할 수 있습니다.

업데이트는 어렵지 않지만, 설치 불가 PC나 오류 상황이 생길 수 있으니 꼭 준비 후 진행하시길 권장합니다.
