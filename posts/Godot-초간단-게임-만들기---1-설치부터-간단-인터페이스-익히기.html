Godot 초간단 게임 개발 일지 1일차! Godot을 처음 배우는 초보자를 위해, 인터페이스를 익히면서 아주 간단한 게임을 만들어 보겠습니다. 이 프로젝트에서는 화면에서 캐릭터를 이동시키는 기본적인 2D 게임을 제작합니다. 1. 새로운 프로젝트 생성하기 Godot 실행 → New Project 클릭 프로젝트 이름: TEST-PROJECT 폴더 경로: 원하는 저장 위치 선택 렌더러 선택: Compatibility (PC & 모바일 모두 실행 가능) Create & Edit 클릭하여 프로젝트 생성 2. Godot 기본 인터페이스 살펴보기 Scene (씬): 게임 화면을 구성하는 요소 Node (노드): 게임을 구성하는 기본 요소 (스프라이트, UI, 물리 등) Inspector (인스펙터): 노드의 속성.......
