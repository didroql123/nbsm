엑셀이 갑자기 실행되지 않을 때 &ndash; Microsoft 365 복구로 해결한 사례
티스토리🗓 2025-10-23

업무 중 갑자기 엑셀이 열리지 않는 현상을 겪은 적이 있습니다.
더블클릭을 해도 아무 반응이 없고,
작업 관리자에서도 프로세스가 전혀 잡히지 않는 상황이었죠.

처음에는 특정 엑셀 파일이 손상된 줄 알았는데,
확인차 파워포인트를 실행해보니 이번엔 에러창이 바로 뜨더군요.
이쯤 되니 “엑셀 문제가 아니라, Microsoft 365 전체 오류다”라는 확신이 들었습니다.

원인 추정

Microsoft 365(오피스 365)는 주기적으로 백그라운드 업데이트가 진행됩니다.
이 과정에서 특정 보안 프로그램이나 윈도우 업데이트와 충돌하면
엑셀, 워드, 파워포인트 모두 실행 불가 상태가 되는 경우가 있습니다.

특히 회사 PC처럼 여러 보안프로그램이 설치된 환경에서는
이런 충돌이 의외로 자주 발생하죠.

해결 방법 – “삭제”가 아니라 “복구”

다행히 이 문제는 완전한 재설치 없이도 해결할 수 있습니다.
저는 다음 순서로 해결했습니다.

  1. 제어판 → 프로그램 및 기능으로 이동
  2. 목록에서 Microsoft 365를 찾기
  3. 우클릭 → 변경 선택
  4. “복구(Repair)” 메뉴 실행

여기서 두 가지 복구 방식이 나옵니다.

  • 빠른 복구: 대부분의 손상 파일 및 충돌을 즉시 복원 (대부분 이걸로 해결)
  • 온라인 복구: 완전 초기화 수준의 복구 (시간 오래 걸림, 재로그인 필요)

저는 빠른 복구만으로 바로 정상화되었습니다.
엑셀, 파워포인트, 워드 모두 다시 잘 열렸습니다.

참고 팁

  • 엑셀만 안 열리더라도, 사실상 Office 전체 손상일 가능성이 높습니다.
  • 삭제/재설치보다 먼저 “복구”를 시도하세요.
  • 보안프로그램 업데이트 직후라면, 해당 프로그램을 일시 중지하고 복구를 진행하면 성공률이 높습니다.
  • 복구 후에는 Windows 업데이트도 최신 상태로 맞춰주는 게 좋습니다.

✅ 정리

구분내용
문제 엑셀 및 오피스 앱이 실행되지 않음
원인 Microsoft 365 손상 또는 업데이트 충돌
해결 제어판 → Microsoft 365 → 변경 → 복구 실행
비고 빠른 복구로 대부분 해결 가능

한 줄 요약:

엑셀이 안 열린다면 삭제보다 “복구”가 먼저입니다.
