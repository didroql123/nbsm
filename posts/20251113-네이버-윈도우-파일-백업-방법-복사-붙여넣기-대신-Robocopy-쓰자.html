윈도우 파일 백업 방법 복사 붙여넣기 대신 Robocopy 쓰자
네이버🗓 2025-11-13
안녕하세요! 윈도우서버의 파일정리를 매번 복사 붙여넣기(ctrl+c / ctrl+v) 로 작업하고, 작업일지쓰고 하던 중 로그도 남길수 있는 방법을 찾았습니다. 윈도우에는 이미 백업용 도구가 있더군요. 이걸왜 지금알았을까... 파일 변경분만 빠르게 옮겨주고, 끊겼던 작업도 이어받을수 있고 해서 좋은 방법인것같아 들고왔습니다. Robocopy 는 단순한 복사가 아니다 robocopy (Robust File Copy) 는 Windows에 기본 포함된 프로급 파일 복사/백업 도구입니다. 일반 복사처럼 단순히 파일을 옮기는게 아니라, 변경된파일만 감지해서 빠르게 백업하고, 끊어지면 이어붙이고, 로그까지 남겨주는 스마트 복사 툴입니다! 사용 예시 제 사무실 서버 기준.......
