Godot 엔진 설치 방법
🗓 2025-02-17
1. Godot 다운로드 Godot 엔진은 공식 웹사이트에서 무료로 다운로드할 수 있습니다. 공식웹사이트의 다운로드 페이지(https://godotengine.org/)로 이동합니다. Godot Engine을 다운 받습니다. 2. Godot 설치하기 Windows 설치 방법 다운로드한 .zip 파일을 압축 해제합니다. godot_v4.3-stable_win64.exe 실행 파일을 더블 클릭하여 실행합니다. 추가 설치 과정 없이 바로 실행됩니다. 3. Godot 실행 및 프로젝트 생성 Godot를 실행하면 처음 화면에서 만들기 버튼을 클릭합니다. 2. 프로젝트 이름과 저장할 경로를 설정하고 만들기&편집을 클릭합니다.
